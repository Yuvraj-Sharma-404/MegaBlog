📝 Mega Blog

«A modern and responsive blogging platform where users can create, publish, and share blogs with images effortlessly.»
---

✨ Features

- 📝 Create and publish blog posts
- 🖼️ Upload images with your blogs
- ✏️ Rich text writing experience
- 📱 Fully responsive design
- ⚡ Fast and clean UI
- 🔒 Secure authentication
- 🎨 Modern and user-friendly interface

---

🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- React.js
- Appwrite
- Tailwind CSS
- Vite

---

🚀 Installation

Clone the repository

Go to the project directory

cd mega-blog

Install dependencies

npm install

Start the development server

npm run dev
